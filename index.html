Agentic Memory

DeepMem: Deep Memory Passing for LLM Agents

Proposed DeepMem to address stale memory caused by single-hop updates and shallow retrieval in long-horizon LLM agents. DeepMem performs multi-hop write-time memory updates and read-time graph propagation without adding query-time LLM calls.

+15.7% F1 on Qwen2.5-1.5B +58.0% F1 on Qwen2.5-3B 70.60 Overall on LongMemEval-S
